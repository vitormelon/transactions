# Transactions #

### Prerequisites

You need `Docker` and `docker-compose`. Install docker compose [documentation](https://docs.docker.com/compose/install/)

### Commands
Run application with docker on port `:8080`

```shell
make up
```

Show and attach to application logs

```shell
make logs
```
